# Info
ERS is an algorithm to Approximate the Number of k-Cliques in Sublinear Time

# To compile
`g++ "-std=c++11" -O3 -o ERS ERS.cpp`

# To run
`./ERS k filepath ϵ δ`
* **ERS** executable file
* **filepath** input file path
* **ϵ** the main approximation parameter 0 < ϵ < 1
* **δ**  the failure parameter
